## 🚀 COMEÇAR AGORA - 3 PASSOS

### 1. Instalar (30 segundos)
```bash
# Windows
setup.bat

# Linux/Mac
chmod +x setup.sh
./setup.sh
```

### 2. Configurar (2 minutos)
- Abra `.env`
- Copie sua Google API Key de: https://makersuite.google.com/app/apikey
- Cole em: `GOOGLE_API_KEY=`

### 3. Rodar (30 segundos)
```bash
python app.py
# Acesse http://localhost:5000
```
